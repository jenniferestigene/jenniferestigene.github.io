Wave Pattern

A grid of oscillating circles creating synchronized wave-like motion

Tools p5.js, JavaScript Type Generative Art, Grid Animation Grid Size 15 × 15 cells (225 particles)

Overview

This sketch creates a mesmerizing wave pattern through the coordinated motion of particles arranged in a grid. Each particle orbits around its fixed center point, with phase offsets between particles creating the illusion of waves propagating across the canvas. The result is a hypnotic, organic motion from simple circular mathematics.

Technical Implementation

  • Framework: p5.js (JavaScript graphics library)
  • Grid Structure: 15 × 15 array of Cell objects
  • Cell Behavior: Each cell orbits around its center in circular motion
  • Phase Offset: Angle = (i × 100) + (j × 100) creates wave propagation
  • Angular Velocity: 0.05 radians per frame for smooth rotation
  • Position Calculation: x = r × cos(angle), y = r × sin(angle)
  • Canvas Size: 400 × 400 pixels with responsive cell sizing
  • Update Loop: Synchronized animation through draw() function
  • Source: Patt Vira
