Slime Molds (Physarum)

An agent-based system inspired by the foraging behavior of Physarum polycephalum

Tools p5.js, JavaScript Type Agent-Based Simulation Agents 4,000 autonomous particles

Overview

This simulation recreates the emergent behavior of slime molds (Physarum Polycephalum) through thousands of autonomous agents. Each agent follows simple rules: sensing its environment through three forward-looking sensors and adjusting its heading based on chemical trails left by other agents.

Despite these simple individual behaviors, the collective system produces complex, organic patterns that mirror the efficient network structures found in biological slime molds. The resulting visual resembles branching neural networks, river systems, or mycelial growth.

Biological Inspiration

Physarum polycephalum is a single-celled organism that exhibits remarkable problem-solving abilities. It can find the shortest path through a maze, optimize transport networks, and even recreate efficient subway systems. All without a brain or nervous system.

The organism moves by extending pseudopods in multiple directions, depositing chemical markers as it travels. It reinforces successful paths (those leading to food) while abandoning unsuccessful ones, creating an efficient network that balances exploration and exploitation.

Algorithm Details

Each agent has three sensors positioned ahead at 45° angles. At each frame, the agent samples pixel brightness at these sensor positions. If the forward sensor detects the highest concentration, the agent continues straight. If the left or right sensor detects more, it turns in that direction. When both side sensors detect equal values, the agent randomly chooses a direction.

This simple decision tree, executed thousands of times per frame across thousands of agents, creates the complex emergent patterns visible in the simulation. The background fade acts as a decay function, preventing trail saturation while maintaining path history.

Technical Implementation

  • Framework: p5.js (JavaScript graphics library)
  • Agent System: 4,000 autonomous particles with individual state
  • Sensing: Three-sensor system (left, forward, right) at 45° angles
  • Movement: Agents turn toward higher chemical concentrations
  • Trail Decay: Background fade creates persistent but fading trails
  • Edge Handling: Toroidal wrapping (agents wrap around canvas edges)
  • Pixel Sampling: Direct pixel array manipulation for chemical sensing
